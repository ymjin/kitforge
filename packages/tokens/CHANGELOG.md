# @kitforge/tokens

## 0.1.0

### Minor Changes

- 3b821c7: Initial release of `@kitforge/tokens` — the single source of truth for the
  kitforge design system. One TS definition (color, spacing, border/radius,
  typography, shadow) emits three outputs: TS/JS exports, CSS custom properties
  (`@kitforge/tokens/css`), and a Tailwind preset (`@kitforge/tokens/tailwind`).
  Consumed by `@kitforge/ui` (web) and `@kitforge/ui-native` (NativeWind).
